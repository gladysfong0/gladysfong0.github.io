WMATA Safety Data Architecture & Forecasting

Python Nonlinear Regression Data Architecture GIS MBSE

Capstone fellowship with GWU × WMATA. Redesigned WMATA's siloed safety data into a centralized logical data architecture for cross-departmental incident analysis across rail and bus operations. Built a Python forecasting tool modeling seasonality and location-based incident patterns for predictive maintenance planning. Led weekly stakeholder meetings with WMATA safety engineers across a 4-person team.

IoT Plant Health Monitoring System

Raspberry Pi 4 Machine Learning SysML Capella Python

Designed and deployed an embedded smart monitoring system with 4 environmental sensors, Raspberry Pi 4, and automated pump control to automate plant health tracking. Developed an ML-based plant health classification model and real-time dashboard. Modeled full system lifecycle using SysML and Capella for design traceability and verification.

FEMA Equitable Disaster Resource Allocation Tool

Python GIS Vulnerability Index Data Analytics

Built a Python tool normalizing socioeconomic, geographic, and infrastructure indicators into a Vulnerability Index to identify Houston-area communities most at risk of inadequate disaster response using Hurricane Harvey data. Produced geospatial heat maps and presented data-driven recommendations to FEMA Response leadership.

E-Scooter Infrastructure Requirements Analysis

GIS Python Requirements Engineering Urban Mobility

Conducted full requirements analysis for ADA-compliant e-scooter docking infrastructure in Foggy Bottom, DC, addressing last-mile transit gaps. Built GIS heat maps visualizing ridership demand and accessibility gaps; translated stakeholder-driven requirements into infrastructure specifications for urban mobility planning.

Mechanical Bird — Systems Design & Fabrication

SOLIDWORKS AutoCAD SysML Machining

Designed and manufactured a mechanical bird with flapping wings and 360-degree head rotation using SOLIDWORKS, AutoCAD, and machining tools. Applied SysML decomposition and iterative design verification to validate functional requirements and ensure structural integrity at each fabrication stage.
